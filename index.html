TEOSC: Tensor Operation Static Counter

TEOSC (Tensor Operation Static Counter) is a Python tool designed to analyze and count PyTorch tensor operations in Python files, Jupyter notebooks, or entire repositories. It leverages Python’s ast (Abstract Syntax Tree) module to statically parse code, detect tensor-related operations without executing the code, and provide insights into model complexity and performance at the code level.
